# Results

The public benchmark table will be synchronized with the final three-seed paper artifacts. This repository does not present preliminary single-seed values as final evidence.

The release protocol compares the promoted Fields model, its no-PCAF-from-initialization ablation, a parameter-matched PCAF-Conv control, a Flash-SDPA Transformer with YaRN, and official Mamba-2 under a paired PG-19 token budget. Long-context scoring covers 2K, 8K, 16K, 32K, and 64K.

When the manuscript is posted, this page will contain:

- mean and standard deviation across three seeds;
- validation and test perplexity;
- context-by-context long-range scores;
- throughput and memory measurements;
- links to generated summaries and the paper table.
